# Frontend Aesthetics Configuration

## Typography

- **Unique & Distinctive**: Avoid generic fonts like Arial, Inter, Roboto, or standard system fonts.
- **Selection**: Choose beautiful, unique fonts that elevate the design (e.g., 'Syne', 'Outfit', 'Clash Display', or 'Cabinet Grotesk').

## Color & Theme

- **Cohesive Aesthetic**: Use CSS variables for consistency.
- **Sharp Accents**: Prefer dominant colors with sharp, high-contrast accents over timid, evenly-distributed palettes.
- **Inspiration**: Draw from IDE themes and cultural aesthetics.

## Motion

- **CSS-Only First**: Prioritize CSS-only solutions for simple interactions.
- **Motion Library**: Use `framer-motion` for complex React animations.
- **High-Impact Moments**: Focus on choreographed page loads with staggered reveals (`animation-delay`) to create delight.

## Backgrounds

- **Depth & Atmosphere**: Avoid solid colors. Layer CSS gradients, geometric patterns, or contextual SVG effects.
- **Contextual**: Backgrounds should match the specific character of the page.
